# Weekly Changelog App

A simple, elegant app for tracking and managing weekly updates.

## Features

- Create timestamped changelog entries
- Edit existing entries with edit history tracking
- Delete unwanted entries
- Persistent storage using localStorage
- Responsive, clean design using Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (version 14 or newer)
- npm or yarn

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/changelog-app.git
cd changelog-app
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

4. Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

## Project Structure

```
changelog-app/

│   ├── components/   # React components
│   ├── styles/       # CSS styles
│   ├── utils/        # Utility functions
│   ├── App.jsx       # Main App component

```

## Customization

- Edit `src/styles/index.css` to customize the app's appearance
- Modify components in the `src/components/` directory to change functionality
